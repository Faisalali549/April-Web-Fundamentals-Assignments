Diffrence between Var and Let

Var

  1. var is function scoped.
  2. var allows to redeclare variables.
  3. Hoisting occurs in var.

Example

function greet() {
// variable a can be used here
var a = "hello"; console.log(a); }
// variable a cannot be used here
greet(); // hello

Let

  1. let is block-scoped.
  2. let does not allow to redeclare variables.
  3. Hoisting does not occur in let.

Example

function greet() {
let a = "hello";
// variable b cannot be used here
if (a == "hello") {
// variable b can be used here
let b = "world"; console.log(a + " " + b); }
// variable b cannot be used here console.log(a + " " + b);
// error }
// variable a cannot be used here
greet();
